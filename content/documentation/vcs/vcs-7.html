--- title: VCS Chapter 7 index: 7 --- {% extends "docs.j2" %} {% block container %} ## CHAPTER 7 VCS Secondary Objects Secondary objects help to specify and define primary objects in VCS. The list of secondary objects include: colormap, fill area, format, line, marker, list, text-combined, text-orientation, and text-table. Note, although the colormap is considered a secondary object, it is acessed differenly. Therefore, the colormap object will not be described below. See the [VCS Command Reference Guide](vcs-5.html) for colormap commands. **Note**, to see the list of object attributes, use the list() function. For example: ``` python import vcs a=vcs.init() ln=a.createline(`new') ln.list() # This call will list the line attributes and their values tt=a.createtexttable(`new') tt.list() # This call will list the text-table attributes and their values ```

Attributes (or Members)

Purpose

Expected Type(s) (or Values)
Object: fillarea Describes the appearance of a block of color. Class Name: Tf
name Used to retrieve this object from VCS StringType
x Used to define the x coordinates of the vertices of the fillarea. List of floats ([0, 1]). Needs to be the same length as y.
y Used to define the y coordinates of the vertices of the fillarea. List of floats ([0, 1]). Needs to be the same length as x.
color The color of the block being filled. IntType: (Range 0 to 255)
Object: line Describes the appearance of a line Class Name: Tl
name Used to retrieve this object from VCS StringType
type Changes the drawing method used for the line to the type specified. Either: 'solid' (0), 'dash' (1), 'dot' (2), 'dash-dot' (3), 'long-dash' (4)
width The width of the line. IntType: (Range 1 to 300)
color The color of the line. IntType: (Range 0 to 255)
Object: marker Describes the appearance of a marker. Class Name: Tm
name Used to retrieve this object from VCS StringType
type Changes the image displayed for this marker to the type specified. Either: "dot" (1), "plus" (2), "star" (3), "circle" (4), "cross" (5), "diamond" (6), "triangle_up" (7), "triangle_down" (8), "triangle_left" (9), "triangle_right" (10), "square" (11), "diamond_fill" (12), "triangle_up_fill" (13), "triangle_down_fill" (14), "triangle_left_fill" (15), "triangle_right_fill" (16), "square_fill" (17), or "hurricane" (18).

Object: textcombined Provides a complete description of text to be rendered. Class Name: Tc
name Used to retrieve this object from VCS StringType
font The font to use for the text. Either: "default" (1), "Clarendon" (2), "Courier" (3), "Helvetica" (4), "Adelon" (5), "Times" (6), "Arabic" (7), "Chinese" (8), "Greek" (9), "Hebrew" (10), "Russian" (11), "Maths1" (12), "Maths2" (13), "Maths3" (14), "Maths4" (15), "AvantGarde" (16)

name Used to retrieve this object from VCS StringType
height The line-height for the text. IntType: (Range 1 to 100)
angle The angle the text will be displayed at. IntType: (Range 0 to 360)

valign The vertical alignment of the text. Either: 'top' (0), 'cap' (1), 'half' (2), 'base' (3), 'bottom' (4)
